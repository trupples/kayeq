Modules
Here is a list of all modules:
 Equalizer moduleThe eq module defines the equalizer model and a minimal controller
 Equalizer math moduleThe eqmath module implements all the processing required to draw the frequency response graphs and to equalize the sound according to the equalizer state
 Sound moduleThe sound module handles dynamically allocated one-channel audio signals, providing functions for in-memory initialisation and copying, as well as loading and storing sounds from and to WAV files
 UI moduleThe ui module provides facilities for displaying terminal graphics to the user and for receiving input from the user
