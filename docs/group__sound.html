Detailed Description

The sound module handles dynamically allocated one-channel audio signals, providing functions for in-memory initialisation and copying, as well as loading and storing sounds from and to WAV files.

All in-memory sounds are expected to have a fixed sample rate, defined by SAMPLERATE. The sound_resample() function enables the conversion of a sound of a different sample rate to the expected one.

If given an already initialised sound, all initialisation functions deallocate the old data before proceeding.
